# Voucher Maker

A small local web app for creating reimbursement vouchers and a single final PDF with receipt supportings.

## Start it

Double-click **Open Voucher Maker.cmd**. The app opens directly in your usual browser; there is no server to start and no address to enter.

## How it works

1. Add an expense, client name, amount, account head and an optional receipt image/PDF. The exact date and time are saved automatically.
2. Repeat for each reimbursement.
3. Enter your name in **Pay to**, check the voucher details, then choose **Generate final voucher PDF**.

The generated PDF begins with a cash/bank voucher formatted like the supplied example and appends every receipt attached during the current session. Basic expense details remain on the device after a refresh; add receipts again if you refresh before generating the final PDF.
